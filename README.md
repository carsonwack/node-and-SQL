# node-and-SQL



Command-Line Startup Instructions:
----------------------------------
1. Execute commands in schema.sql  (sets up database and table)

2. Execute commands in seeds.sql   (populates table)

3. git clone https://github.com/carsonwack/node-and-SQL.git

4. cd node-and-SQL

5. node bamazon.js

6. run customerview
